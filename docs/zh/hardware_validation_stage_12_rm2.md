# Stage 12 RM2 Baseline 验证

本清单独立于 Piper 验证，也不是 RM2 运动授权。任何可能运动的步骤之前，必须完成
RM2 专属姿态/回放安全门禁、工作区清理、关节单位复核和急停验证。

1. 在黑相机且不启动 episode 的条件下创建 Baseline；核查 RM2 ActionSpec、相机
   角色、策略关节单位和运行时元数据。
2. 只克隆一个实验变量，并审查 Baseline diff。
3. 创建由 Baseline 支持的评测，确认它只进入人工准备阶段；操作员明确开始一轮前
   不会发送动作。
4. 获授权后先做 3–5 轮冒烟、10 轮初筛，再每种配置做 20–30 轮比较。保留
   invalid、timeout 和 safety abort 计数，不得隐去。

MockArm 通过不等于真机验证。出现意外运动或通信故障时，使用已验证的 RM2 物理
急停或紧急流程。

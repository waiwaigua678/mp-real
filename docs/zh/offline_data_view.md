# 离线数据查看

`mp-data-view` 是录制数据查看器。普通浏览只读取文件，不会创建 Robot、相机或策略资源，
所以可与相机预览共存。

`mp-real-web` 的 `DATA_VIEW` 资源模式将同一查看器嵌入“运行”页。可在启动时传入
`--recorded-data-root <目录>`，也可选择 `DATA_VIEW` 后在页面的“导入数据路径”中输入运行
Web 服务那台机器可读取的目录；导入只在当前 Web 进程有效，不会写入磁盘配置。随后即可查看
三路相机、状态曲线、录制动作曲线与播放时间轴。

- **开环评测**必须显式提交。它把录制的图像和状态 teacher-forced 地发送给策略服务，绘制
  预测动作（实线）与录制目标动作（虚线），但绝不创建 Robot 或下发动作。
- **真机回放**是另一条路径：先离线生成并验证安全计划，再连接机器人、低速移动到起点，
  最后需要二次确认才会运动。

因此，数据查看本身不是录制状态移动、部署交接或真机评测；不能把“开环结果可见”视为
真机行为已验证。
